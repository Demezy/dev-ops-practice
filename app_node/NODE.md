# description 

Node js app that gives date in moscow time zone

# installation

feel free to use any js package manager. For an example `pnpm` would be used. Assume that you are in `app_node` folder.
Make sure `node` installed.

```bash
pnpm i
pnpm exec tsc
node src/server.js
```

# testing

```bash
pnpm exec jest
```
